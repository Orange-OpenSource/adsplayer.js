addElement

The 'addElement' event is fired when a DOM element for playing an ad has been created and will be appended in the ads player container. The element can be either a <video> or an <img> element.

Parameters:
Name Type Description
event object

the event

Properties
Name Type Description
type object

the event type ('addElement')
data object

the event data

Properties
Name Type Description
element string

the created element
type string

the type of the element, 'video' for <video> or 'image' for <img>
